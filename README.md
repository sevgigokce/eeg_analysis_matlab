# EEG-Based Imagined Swallowing Analysis (MATLAB)

This repository contains the custom MATLAB scripts used in the study entitled:

**“EEG-Based Electrophysiological Analysis of Imagined Swallowing”**

The code supports EEG preprocessing, feature extraction, model training, and performance evaluation for imagined swallowing classification tasks.


## Repository Structure

  Scripts for EEG signal preprocessing (e.g., segmentation, normalization).
  Feature computation routines used in the study.
  Classification and model training scripts.
  Performance evaluation, metrics computation, and cross-validation procedures.


## Requirements

- MATLAB R2021b or later  
- Signal Processing Toolbox  
- Statistics and Machine Learning Toolbox  


## Usage

1. Clone or download this repository.
2. Open MATLAB and set the repository directory as the working path.
3. Update data paths in the relevant scripts.
4. Run the main analysis or model training scripts as described in the manuscript.
